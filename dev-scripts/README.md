# Development Scripts

This folder contains PowerShell scripts for development tasks.

## Available Scripts

- **build-android-app.ps1**: Build Android TWA app using Bubblewrap
- **cleanup.ps1**: Clean up build artifacts and temporary files
- **clear-cache.ps1**: Clear Next.js cache and rebuild
- **fix-hydration-errors.ps1**: Fix React hydration errors
- **restart-dev.ps1**: Restart development server

## Usage

Run scripts from the project root:
```powershell
./dev-scripts/build-android-app.ps1
./dev-scripts/cleanup.ps1
./dev-scripts/restart-dev.ps1
```
