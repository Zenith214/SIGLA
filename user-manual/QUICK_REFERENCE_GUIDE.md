# PULSE Quick Reference Guide

**Version 2.0** | **January 7, 2026**

---

## 🚀 Quick Start by Role

### Admin
1. **Login** → Settings
2. **Create Survey Cycle** → Set as Active
3. **Add Barangays** → Set Targets
4. **Create Users** → Assign Roles
5. **Assign Supervisors** → Monitor Progress

### Field Supervisor
1. **Login** → View Assigned Barangays
2. **Create Spots** → Assign Interviewers
3. **Monitor Progress** → Check Data Quality
4. **Support Interviewers** → Report Issues

### Interviewer
1. **Login** → Survey Page
2. **Select Spot** → Start Survey
3. **Use Kish Grid** → Complete Sections
4. **Geotag Location** → Submit
5. **Sync Offline Surveys** (if applicable)

### Officer
1. **Login** → View Dashboard
2. **Review Report Card** → Analyze Results
3. **Create CPAP** → Add Action Items
4. **Use AI Suggestions** → Submit for Review
5. **Update Progress** (after approval)

### Viewer
1. **Login** → View Dashboard
2. **Explore Map** → Click Barangays
3. **View Analytics** → Export Data
4. **Read Report Cards** → Monitor Progress

## 🔑 Key Features at a Glance

### Dashboard
- **Hover**: Quick preview of barangay data
- **Click**: Lock selection for detailed view
- **Toggle**: Switch between Map and List view
- **Cycle Selector**: View historical data

### Survey (Interviewers)
- **6 Service Areas**: Financial, Disaster, Social, Safety, Business, Environmental
- **Kish Grid**: Random respondent selection
- **Geotagging**: GPS location capture
- **Offline Mode**: Work without internet
- **Auto-Sync**: Automatic upload when online

### CPAP (Officers)
- **13 Columns**: Comprehensive action planning
- **AI Suggestions**: ML-powered recommendations
- **Comments**: Admin-officer communication
- **Status Flow**: Draft → Submitted → Approved
- **Progress Tracking**: Implementation monitoring

## ⚡ Common Tasks

### Create Survey Cycle (Admin)
Settings → Survey Cycles → Create New Cycle → Fill Form → Save → Set as Active

### Assign Supervisor (Admin)
Settings → Supervisor Assignments → Create Assignment → Select FS & Barangay → Save

### Create Spot (FS)
Settings → Spots → Create Spot → Fill Details → Assign Interviewer → Save

### Conduct Survey (Interviewer)
Survey → My Spots → Select Spot → Start Survey → Kish Grid → Complete → Geotag → Submit

### Create CPAP (Officer)
CPAP → Add Item → Select Service Area → Fill Required Fields → Save → Submit for Review

### Review CPAP (Admin)
Admin → CPAP Review → Select CPAP → Review Items → Add Comments → Approve/Request Revision

## 🔧 Quick Troubleshooting

| Problem | Quick Fix |
|---------|-----------|
| Can't login | Check email/password, verify account active |
| Dashboard blank | Refresh page, check active cycle |
| Survey won't save | Check internet, save as draft first |
| GPS not working | Enable location, move outdoors, wait 30-60s |
| Offline surveys not syncing | Stable internet, don't close browser, refresh |
| Can't submit CPAP | Fill required Output column, check status is Draft |
| Data not updating | Refresh page (F5), clear cache |
| Slow performance | Close tabs, clear cache, check internet speed |

## 💡 Pro Tips

**For Interviewers**:
- Aim for 8-12 surveys/day
- Use offline mode in remote areas
- Verify GPS accuracy < 20m
- Schedule callbacks for unavailable respondents

**For Officers**:
- Review report card before creating CPAP
- Use AI suggestions to save time
- Prioritize critical issues first
- Update progress monthly

**For Admins**:
- Export data weekly for backup
- Monitor survey progress daily
- Review CPAPs within 3 days
- Set realistic targets based on MOE

**For Field Supervisors**:
- Create spots covering entire barangay
- Distribute targets evenly
- Check data quality daily
- Support interviewers proactively
