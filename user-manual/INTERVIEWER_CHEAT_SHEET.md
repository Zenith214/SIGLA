# Interviewer Cheat Sheet - PULSE System

**Quick Reference for Field Interviewers**

---

## 🎯 Your Responsibilities

✅ Conduct face-to-face surveys with citizens  
✅ Use Kish Grid for respondent selection  
✅ Collect accurate and complete data  
✅ Geotag survey locations  
✅ Submit surveys (online or offline)  
✅ Meet daily/weekly quotas  

---

## 🚀 Daily Workflow

### Morning Routine
1. **Login** to PULSE system
2. Check **My Spots** tab
3. Review today's target
4. Ensure device is charged
5. Enable GPS/location services
6. Check internet connection

### During Fieldwork
1. Navigate to assigned spot
2. Select household
3. Use Kish Grid to select respondent
4. Conduct survey (20-30 minutes)
5. Geotag location
6. Submit or save as draft
7. Move to next household

### Evening Routine
1. Submit all draft surveys
2. Sync offline surveys
3. Review progress
4. Plan tomorrow's route
5. Charge devices

---

## 📋 Survey Process - Step by Step

### Step 1: Start Survey
```
Survey Page → My Spots → Select Spot → Start New Survey
```

### Step 2: Kish Grid Selection
```
1. Ask: "How many people aged 18+ live here?"
2. Enter number in system
3. List all eligible members (oldest to youngest)
4. System selects ONE person
5. Interview ONLY that person
```

**Important**: Never substitute! If unavailable, schedule callback.

### Step 3: Complete 6 Service Areas

**For Each Service Area**:
1. **Awareness**: "Are you aware of [service]?" (Yes/No)
2. **Availment**: "Have you used [service]?" (Yes/No) - *Skip if not aware*
3. **Satisfaction**: "How satisfied?" (1-5 scale) - *Skip if not used*
4. **Need for Action**: "Does this need improvement?" (Yes/No)
5. **Suggestions**: Open-ended feedback

**Service Areas**:
1. Financial Administration
2. Disaster Preparedness
3. Social Protection
4. Safety and Peace
5. Business-Friendly
6. Environmental Management

### Step 4: Geotagging
```
1. Click "Get Current Location"
2. Allow location access
3. Wait for GPS signal (30-60 seconds)
4. Verify accuracy < 20 meters
5. Confirm location
```

### Step 5: Submit
```
1. Review all answers
2. Check required fields (marked *)
3. Add remarks if needed
4. Click "Submit" (or "Save as Draft")
```

## 🗺️ Kish Grid Quick Reference

| Household Size | Selection Pattern |
|----------------|-------------------|
| 1 person | Select that person |
| 2 people | Alternate (1st, 2nd, 1st, 2nd...) |
| 3 people | Rotate (1st, 2nd, 3rd, 1st...) |
| 4+ people | System calculates |

**System handles this automatically - just list members by age!**

## 📱 Offline Mode Guide

### When to Use
- Poor internet connection
- Remote areas
- To save mobile data

### How It Works
1. Surveys save locally in browser
2. "Offline" indicator appears
3. Continue surveying normally
4. Auto-syncs when connection restored

### Important Rules
⚠️ Don't clear browser cache while offline  
⚠️ Sync as soon as connection available  
⚠️ Check for "sync complete" notification  
⚠️ Don't close browser during sync

## 🎯 Daily Targets

**Recommended**: 8-12 surveys per day

**Time Breakdown**:
- Survey: 20-30 minutes
- Travel between households: 10-15 minutes
- Breaks: 1-2 hours total
- **Total**: 6-8 hours fieldwork

**Tips for Meeting Targets**:
- Start early (8-9 AM)
- Work during peak hours (evenings/weekends)
- Plan efficient routes
- Schedule callbacks for unavailable respondents
- Use offline mode to save time

## ✅ Quality Checklist

Before submitting each survey:

- [ ] Correct respondent selected via Kish Grid
- [ ] All required fields completed (marked *)
- [ ] Answers are consistent (no contradictions)
- [ ] GPS coordinates captured (accuracy < 20m)
- [ ] Location is within assigned spot/barangay
- [ ] Survey completed in reasonable time (15-45 min)
- [ ] Respondent's feedback accurately recorded
- [ ] No duplicate surveys for same household

## 🚫 Common Mistakes to Avoid

❌ **Substituting respondents** - Always use Kish Grid selection  
❌ **Rushing through surveys** - Take time for accurate responses  
❌ **Leading questions** - Stay neutral, don't influence answers  
❌ **Skipping geotagging** - Always capture location  
❌ **Ignoring skip patterns** - Follow conditional logic  
❌ **Submitting incomplete surveys** - Check all required fields  
❌ **Interviewing same household twice** - Check for duplicates  
❌ **Working outside assigned spots** - Stay in your area  

---

## 📍 Geotagging Tips

### For Best GPS Accuracy
✅ Conduct surveys outdoors when possible  
✅ Wait 30-60 seconds for GPS to stabilize  
✅ Aim for accuracy < 20 meters  
✅ Avoid tall buildings and dense tree cover  
✅ Enable "High Accuracy" mode on device  

### If GPS Not Working
1. Move to open area
2. Restart location services
3. Wait longer for signal
4. Manually enter coordinates (last resort)
5. Note issue in remarks

## 💬 Interview Tips

### Building Rapport
- Introduce yourself and purpose
- Show ID/authorization letter
- Explain confidentiality
- Be friendly and respectful
- Use local language if possible

### Asking Questions
- Read questions exactly as written
- Don't explain or interpret
- Stay neutral (no leading)
- Repeat if needed
- Clarify without influencing

### Handling Difficult Situations
- **Respondent refuses**: Thank them, move to next household
- **Respondent unsure**: Offer "Don't know" option
- **Respondent hostile**: Stay calm, politely end interview
- **Language barrier**: Request translator or reschedule
- **Interruptions**: Pause survey, resume when possible
