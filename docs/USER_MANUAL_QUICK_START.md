# PULSE System - Quick Start Guide

**5-Minute Overview for New Users**

---

## 🎯 What is PULSE?

A digital platform for:
- Collecting citizen feedback on local government services
- Creating data-driven action plans (CPAP)
- Tracking implementation progress
- Improving barangay governance

---

## 👤 Your Role

### Admin
**You**: Set up cycles, assign supervisors, monitor everything
**Start**: Settings → Create Survey Cycle

### Field Supervisor
**You**: Create spots, assign interviewers, monitor data collection
**Start**: Spots → Create New Spot

### Interviewer
**You**: Conduct surveys with citizens
**Start**: Surveys → Start New Survey

### Barangay Official
**You**: Create action plans based on survey results
**Start**: CPAP → Create a Plan

---

## 🚀 Quick Actions

### For Admins
```
1. Create Survey Cycle
2. Award Barangays
3. Set Targets (with MOE)
4. Assign Field Supervisors
5. Monitor Progress
```

### For Field Supervisors
```
1. View Assigned Barangays
2. Create Survey Spots
3. Assign Interviewers
4. Monitor Data Collection
```

### For Interviewers
```
1. View Assigned Spots
2. Start New Survey
3. Use Kish Grid for Selection
4. Complete Questions
5. Submit Response
```

### For Barangay Officials
```
1. View Survey Results
2. Click "Create a Plan"
3. Add Action Items
4. Use AI Suggestions (optional)
5. Submit for Review
```

## 📊 CPAP Spreadsheet Basics

### 13 Columns
1. Results/Observations
2. Plan of Action
3. Activity
4. **Output** (REQUIRED)
5. Actual Output
6. Status
7. Schedule
8. Actual Date
9. Budget
10. Responsible Person
11. Committed
12. Verification
13. Delete

### 6 Service Areas
1. Financial Administration
2. Disaster Preparedness
3. Social Protection
4. Safety and Peace
5. Business-Friendly
6. Environmental Management

### Key Actions
- **Add Row**: Click "+ Add row for [Area]"
- **Delete Row**: Click trash icon
- **Save**: Click "Save All Changes"
- **AI Help**: Click "Get AI Suggestions"

---

## 💡 Pro Tips

1. **Tab Key**: Navigate between cells quickly
2. **Save Often**: Changes aren't auto-saved
3. **AI Suggestions**: Use as starting point, edit to fit your needs
4. **Comments**: Use sidebar to communicate with admin
5. **Notifications**: Check red badge for updates

---

## ⚠️ Common Mistakes

❌ **Don't**: Submit without filling Output column
✅ **Do**: Fill all required fields before submitting

❌ **Don't**: Trust AI suggestions blindly
✅ **Do**: Review and edit all AI-generated content

❌ **Don't**: Delete rows without saving first
✅ **Do**: Save before making major changes

❌ **Don't**: Forget to check notifications
✅ **Do**: Visit CPAP page regularly for updates

---

## 🆘 Quick Troubleshooting

**Can't login?** → Check username/password, clear cache
**Can't see CPAP?** → Verify role and barangay assignment
**Changes not saving?** → Check internet, look for errors
**AI not working?** → Ensure survey data exists, refresh page
