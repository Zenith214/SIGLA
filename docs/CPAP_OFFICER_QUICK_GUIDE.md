# CPAP Officer Quick Guide

## Accessing CPAP

1. Log in as an Officer user
2. Click your profile icon in the top-right corner
3. Select "CPAP Submission" from the dropdown menu
4. Or navigate directly to `/cpap`

## Creating Your First CPAP

### Step 1: Add Action Items

1. Click the "Add Item" button
2. Fill in all required fields:
   - **Priority Area**: The service area or focus (e.g., "Health Services")
   - **Target Output**: What you plan to accomplish
   - **Success Indicator**: How you'll measure success
   - **Responsible Person**: Who will lead this action
   - **Timeline**: Start and end dates
3. Click "Add Item" to save
4. Repeat for all action items

### Step 2: Use AI Suggestions (Optional)

1. Click the "AI Suggestions" button
2. Review AI-generated recommendations
3. Click "Use These Suggestions" to add them
4. Review and edit the suggested items
5. Click "Save All" to add them to your CPAP
6. Or click "Discard" to remove them

### Step 3: Submit for Review

1. Ensure all items have complete information
2. Click "Submit to DILG for Review"
3. Confirm submission in the dialog
4. Your CPAP is now submitted and read-only

## Handling Revisions

If DILG requests revisions:

1. You'll see an amber banner with admin comments
2. All fields become editable again
3. Make the requested changes
4. Click "Resubmit to DILG"

## Tracking Progress

Once your CPAP is approved:

1. The interface switches to Progress Tracker mode
2. Original plan details are shown (read-only)
3. Update these fields for each item:
   - **Actual Output**: What was actually accomplished
   - **Accomplishment Status**: Current status (e.g., "Completed", "In Progress")
   - **Remarks**: Additional notes
4. Click "Save Progress Update" when done

## CPAP Status Meanings

- **Draft**: You can edit freely
- **Submitted**: Under DILG review (read-only)
- **Approved**: You can update progress
- **Revision Requested**: DILG wants changes (editable)

## Tips

- Changes auto-save after 1 second of inactivity
- You can edit or delete items anytime in Draft status
- Use AI suggestions as a starting point, then customize
- Be specific in your success indicators
- Set realistic timelines
- Update progress regularly once approved

## Troubleshooting

**Can't access CPAP page?**
- Ensure you're logged in as an Officer
- Check that you have a barangay assignment
- Verify an active survey cycle is configured

**Can't submit CPAP?**
- Ensure all items have complete information
- Check that all required fields are filled
- Verify date ranges are valid (end after start)

**AI Suggestions not loading?**
- Check your internet connection
- Ensure sufficient survey data exists
- Try refreshing the page

**Changes not saving?**
- Check your internet connection
- Look for error messages
- Try refreshing and re-entering changes

## Need Help?

Contact your system administrator if you encounter issues or need assistance with the CPAP module.
