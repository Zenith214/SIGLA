# CPAP Spreadsheet User Guide

## What's New?

The CPAP (Citizen Priority Action Plan) submission now features an **embedded spreadsheet interface** that makes it easier to create and manage action plans.

## Key Changes

### 1. **"Barangay of:" instead of "Municipality"**
The header now displays "Barangay of: [Barangay Name]" to better reflect the administrative level.

### 2. **Survey Cycle Title**
Instead of "C.Y." (Calendar Year), the interface now shows the full survey cycle name and year (e.g., "Survey Cycle 2024: 2024").

### 3. **Embedded Spreadsheet**
The new spreadsheet interface includes all 13 columns from the official CPAP form:
- Results/Observations on Specific Target Service Area
- Plan of Action
- Activity
- Output
- Actual Output
- Status of Accomplishment
- Implementation Schedule
- Actual Date
- Financial Requirements
- Responsible Person/Office
- Committed/To be Committed in Sectoral Plan/Budget
- Means of Verification
- Actions (Delete)

### 4. **Service Area Organization**
The spreadsheet is organized by 6 service areas:
1. **FINANCIAL ADMINISTRATION**
2. **DISASTER PREPAREDNESS**
3. **SOCIAL PROTECTION**
4. **SAFETY AND PEACE**
5. **BUSINESS-FRIENDLY**
6. **ENVIRONMENTAL MANAGEMENT**

## How to Use

### Creating a New CPAP

1. **Navigate to CPAP Page**
   - Go to `/cpap` from your dashboard
   - If no CPAP exists, you'll see a "Create a Plan" button

2. **Click "Create a Plan"**
   - This opens the spreadsheet editor
   - You'll see 8 service area sections

3. **Fill in Data**
   - Each service area has its own section
   - Click "Add row for [Service Area]" to add entries
   - Fill in the columns as needed
   - Use Tab key to move between cells

4. **Add Multiple Rows**
   - You can add multiple action items per service area
   - Click "Add another row" at the bottom of each section

5. **Delete Rows**
   - Click the trash icon in the Actions column to remove a row

6. **Save Your Work**
   - Click "Save Plan" in the top-right corner
   - Or click "Save All Changes" at the bottom of the spreadsheet

### Editing an Existing CPAP

1. **Navigate to CPAP Page**
   - Go to `/cpap` from your dashboard
   - You'll see your existing CPAP

2. **Click "Edit in Spreadsheet View"**
   - This opens the spreadsheet editor with your existing data

3. **Make Changes**
   - Edit any cell by clicking on it
   - Add or remove rows as needed

4. **Save Changes**
   - Click "Save All Changes" to persist your edits

## Tips

- **Use Tab Key**: Press Tab to quickly move between cells
- **Multi-line Text**: Textarea fields automatically expand for longer content
- **Implementation Schedule**: Use format like "Jan - Dec 2024" or "Q1 2024"
- **Financial Requirements**: Enter amounts with currency (e.g., "50,000.00 PHP")
- **Auto-save**: Changes are saved when you click the Save button

## Column Descriptions

| Column | Description | Example |
|--------|-------------|---------|
| **Results/Observations** | Survey findings or community needs | "Availment of services on the Prevention and Management of Diseases" |
| **Plan of Action** | Proposed solution or intervention | "Acquisition of medical equipment" |
| **Activity** | Specific activity to be performed | "Purchase of Newborn Test Machine" |
| **Output** | Expected deliverable | "Acquisition of 1 Newborn Screening Test Machine" |
| **Actual Output** | What was actually delivered | "New born babies undergone screening" |
| **Status of Accomplishment** | Current status | "To be purchased" / "Completed" / "On-going" |
| **Implementation Schedule** | Timeline for completion | "Jan - Dec 2018" |
| **Actual Date** | When it was actually completed | "July 8, 2018" |
| **Financial Requirements** | Budget needed | "50,000.00" |
| **Responsible Person/Office** | Who is accountable | "MHO" (Municipal Health Office) |
| **Committed/To be Committed** | Budget allocation status | "LDIP" / "AIP/LDIP" |
| **Means of Verification** | How to verify completion | "Attendance Sheet, Patient & Participants" |

## Troubleshooting

**Q: I can't see the spreadsheet**
- Make sure you're logged in as an Officer or Admin
- Check that you're assigned to a barangay
- Verify that an active survey cycle exists

**Q: My changes aren't saving**
- Click the "Save All Changes" button
- Check your internet connection
- Look for error messages at the top of the page

**Q: I accidentally deleted a row**
- If you haven't saved yet, refresh the page to restore
- If you've saved, you'll need to re-enter the data

**Q: How do I go back to the old interface?**
- Click the back arrow in the top-left
- The main CPAP page still shows the list view
